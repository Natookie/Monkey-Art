
# 🙊 Monkey Art - Tools for Pixel Artist

Monkey Art is a web-based toolkit designed to simplify and accelerate the pixel art creation process, wether you're an experienced artist or a game developer with little to no art background.

## ✨What is Monkey Art
Monkey Art is a basic tool platform that provides essential tools for creating, managing, and understanding pixel art colors. It focuses heavily on color workflows, helping users generate visually cohesive palettes and apply them effectively.
## 🔮 Current Features
**🎨 Color Palette Generator**

Generate harmonious color palettes instantly. Designed to help users who struggle with color selection or want a faster starting point.

**💾 Save Palette**
Store and manage your palettes for reuse across projects. No more losing good color combinations.

**🌗 Shading Tools**

Easily create shading variations from base colors. Helps maintain consistency and improves lighting workflows.

**🔀 Color Blending**

Blend colors seamlessly to create gradients, transitions, and intermediate tones — essential for polished pixel art.

**📘 Built-in Color Theory Guide**

A lightweight, beginner-friendly introduction to color theory to help users understand why certain colors work well together.
